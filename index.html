Disclaimers and facts

This predictions are entirely based on available data about progression of known cases around the world. This means that the predictions offerred here are based on how cases have progressed around the world. If there are unique cases that have never been seen before, the model will offer a high degree of uncertainty.

> Disclaimer

The support vector machine for regression (SVR) is trained over the number of cases in proportion to the population of the region in consideration using available census data. Before training, the dataset is z-scored to improve performance.

> Fact

The long-short term memory (LSTM)-based model, as opposed to the SVR, uses data that is not z-scored; however, it does have a batch-normalization layer after the input layer. A small number of units has proven to be enough to give good predictions.
